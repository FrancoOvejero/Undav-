# Política de Privacidad - UNDAV+

**Última actualización:** Julio 2026

## Introducción

UNDAV+ tiene como único propósito mejorar la experiencia de uso del sistema SIU Guaraní de la Universidad Nacional de Avellaneda. Sus funciones están orientadas a facilitar la navegación y consulta de información académica dentro de SIU Guaraní UNDAV.

UNDAV+ no forma parte oficialmente de la Universidad Nacional de Avellaneda ni del sistema SIU Guaraní.

## Datos que utiliza la extensión

La extensión puede leer información visible dentro del sitio `academica.undav.edu.ar` para mostrar funciones como favoritos, modo oscuro, progreso del plan de estudios, calendario académico y búsqueda en oferta de comisiones.

Esta información se procesa localmente en el navegador del usuario.

## Almacenamiento local

UNDAV+ utiliza almacenamiento local del navegador, incluyendo `chrome.storage` y `localStorage`, para guardar preferencias y datos necesarios para el funcionamiento de la extensión, como:

- Estado del modo oscuro.
- Páginas favoritas guardadas por el usuario.
- Datos temporales del plan de estudios, materias y correlatividades para evitar cargas repetidas.

## Datos que no recopilamos

UNDAV+ no recopila contraseñas, datos de pago, información financiera, mensajes personales ni datos ajenos al funcionamiento académico dentro de SIU Guaraní UNDAV.

## Uso de los datos

Los datos se utilizan únicamente para mostrar y mejorar las funciones visibles de UNDAV+ dentro del sistema SIU Guaraní UNDAV.

La extensión no vende, comparte ni transfiere datos personales a terceros.

## Transmisión de datos

UNDAV+ no envía datos personales a servidores propios ni a servicios externos.

La extensión funciona dentro del navegador del usuario y se comunica únicamente con el sitio `academica.undav.edu.ar` cuando es necesario para mostrar información del propio sistema SIU Guaraní.

## Seguridad de los datos

Los datos almacenados por UNDAV+ permanecen en el navegador del usuario. La extensión no tiene servidores propios, no realiza seguimiento publicitario y no utiliza la información del usuario para publicidad personalizada.

## Permisos de la extensión

La extensión solicita los siguientes permisos:

- `storage`: se utiliza para guardar preferencias locales, favoritos y datos temporales necesarios para las funciones de UNDAV+.
- `activeTab`: se utiliza para interactuar con la pestaña activa cuando el usuario está usando SIU Guaraní UNDAV.
- Acceso a `https://academica.undav.edu.ar/*`: se utiliza para que la extensión funcione únicamente dentro del portal SIU Guaraní de UNDAV.

## Servicios de terceros

UNDAV+ no comparte datos personales con servicios de terceros.

La extensión puede incluir enlaces externos, como el calendario académico oficial de UNDAV, pero abrir esos enlaces queda bajo decisión del usuario y se rige por las políticas del sitio correspondiente.

## Derechos del usuario

El usuario puede desinstalar la extensión en cualquier momento para eliminar su funcionamiento del navegador.

También puede borrar los datos locales de la extensión desde la configuración del navegador o eliminando los datos asociados a la extensión.
